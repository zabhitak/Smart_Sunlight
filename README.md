# Features

FoodBooth can :
>
>* 🔐- User Authentication
>* 👻- Admin Dashboard
>* ⚡- Realtime - Action Notifications
>* 📱- OTP Authentication
>* 💰- Device Management System
>* ⭐- Realtime Status Update functionality using Sockets.
>* 📩- Email Sending On Register & Order Placement.
>* 🌎- Promotional Emails
